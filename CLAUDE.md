# AI 비주얼라이즈 원데이 강의 프로젝트

## 프로젝트 개요
- **강의명**: "당신의 보고서를 바꿀 AI 비밀무기 3가지"
- **강사**: AICLab 김진수
- **도구**: 노트북LM × 제미나이 × 클로드 × 안티그래비티
- **과정 구성**: 원데이 5시간 / 2일×3시간 / 모객 특강 2시간

## 핵심 파일 구조
```
docs/
  lecture-plan/
    core-brief.md          ← 강의 기획 브리프 (마스터 문서)
    promo-kit.md           ← 홍보 키트 (SNS 카피, 포스터, 이메일)
    promo-kit.docx         ← 홍보 키트 Word 버전
    landing-page-plan.md   ← 상세페이지 기획안 (10섹션)
    landing-page.html      ← 상세페이지 HTML (반응형)
  session-log-2026-03-13.md ← 첫 세션 작업 로그
scripts/
  generate-docx.js         ← promo-kit.docx 생성 스크립트
```

## 작업 규칙
- 마스터 문서는 `core-brief.md` — 커리큘럼 변경 시 여기를 먼저 수정
- 홍보 자료 수정 시 promo-kit.md → docx → landing-page.html 순서로 동기화
- 날짜/요일: 모객 특강은 **토요일** (수정본 기준)
- Canva 디자인 도구 사용하지 않기
